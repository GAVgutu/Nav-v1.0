目前运行在服务器上的环境版本是：
- `PHP 7.4.30`
- `Apache 2.4.54`
- `MySQL 5.6.50`

网站不采用任何框架开发，代码包含很多个人元素，可能会因为环境导致安装失败。

相信也不会有人使用我的程序，因为太久没有编写我已经忘记了最初的代码结构以至于现在我也不知道我之前写的是什么。
